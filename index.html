Hoe maak ik een HTML/CSS bestand

Maak een nieuw bestand in Visual Studio Code

Maak een nieuw bestand aan door op "file/create new file" te klikken. 

Sla dit bestand op

Sla vervolgens je bestand op als "index.html".

Defineer het bestand als HTML

defineer het bestand door de volgende code toe te voegen.
<!DOCTYPE html> <html> </html>

Voeg de content van je website toe 

Denk hierbij aan dingen zoals Head, body, kopjes, paragrafen. Zorg er ook voor dat je indenting (tab)gebruikt voor een overzichtelijk document

Maak je CSS bestand aan

Maak vervolgens nog een bestand aan in Visual Studio Code en sla deze op als style.css op dezelfde locatie als je index.html

Link je CSS met je HTML

Link je CSS met je HTML bestand door de volgende code in je head te plaatsen.
<link rel="stylesheet" href="style.css">

Style je HTML

Gebruik vervolgens de Selectoren in CSS om specifieke elementen vorm te geven in je HTML. Denk hierbij aan kleur/achtergrondkleur, fonts, schaduwen, etc. 

Bekijk je website in de browser

Bekijk je website door het HTML bestand te openen met een internetbrowser (Chrome, Safari, Firefox). Blijf vervolgens verder werken aan je HTML en CSS totdat je tevreden bent.
